### How to run?

```bash
npm i
npm run dev
```

### [Documentation](public/docs/index.html)

Documentation was generated by TypeDoc.

```bash
npm run docs
```

### Rules

- If you don't get the correct answer in 6 tries, the word doesn't change.
- Otherwise, the word changes.

### For Reviewers
